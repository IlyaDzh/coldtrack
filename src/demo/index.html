This is a Live Demo showcasing the essential features of the ColdStack Storage Service. Your wallet MUST be unlocked before uploading any files. Please note that the demo works on the RINKEBY TESTNET, and you must set your wallet to this network accordingly.

Upon unlocking your wallet, you will receive 1.00 testnet $CLS token for the demo. The service will charge you this testnet token, and then you can upload and download any files, and browse storage transactions.

Please note our demo only supports files up to 3GB in size. You can only download files you have uploaded earlier. Access permissions and file sharing are disabled in demo mode.

Please feel free to contact us if you have any questions, and check out our step-by-step guide on Medium: https://medium.com/coldstack/coldstack-live-demo-what-you-can-do-right-now-bfbb94a1bcd0
